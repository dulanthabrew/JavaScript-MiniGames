# JavaScript Mini_Games Collection

A collection of interactive mini-games built using HTML, CSS (Bootstrap 5), and vanilla JavaScript. This project features a central dashboard that allows users to easily navigate and play different games.

## ğŸ® Included Games

### 1. Memory Game
Test your memory by matching pairs of cards. This version features car brand logos as the card faces.
- **Location:** `./memory-game/`
- **Key Features:** Card flipping animation, match logic, move counter.

### 2. Speed Typing Game
Challenge your typing speed and accuracy. Type the displayed words as fast as you can before the time runs out.
- **Location:** `./speed-typing-game/`
- **Key Features:** Random word generation, timer, score tracking, difficulty levels.

### 3. Rock Paper Scissors
The classic hand game played against the computer.
- **Location:** `./rock-paper-scissors-game/`
- **Key Features:** Interactive UI, random computer choice logic, score tracking.

## ğŸš€ Getting Started

### Prerequisites
You need a modern web browser to run these games. No additional software or installation is required.

### Installation & Usage
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/dulanthabrew/JavaScript-MiniGames.git
    ```
    *(Or download the ZIP file and extract it)*

2.  **Open the project:**
    Navigate to the project folder and open `index.html` in your web browser.

3.  **Play:**
    -   Enter your name on the welcome screen.
    -   Click "Enter" to go to the Dashboard.
    -   Select a game card to start playing!
